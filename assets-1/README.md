# Place your background music file here

Add a file named **`music.mp3`** in this `assets/` folder to enable the
background music toggle (the floating speaker button on the page).

- Recommended: a short, soft instrumental track that loops well.
- Keep the file small (ideally under ~3 MB) for fast loading on mobile.
- The audio starts **muted** and only plays after the visitor taps the
  speaker button (this respects browser autoplay rules).

If no `music.mp3` is present, the page still works perfectly — the speaker
button simply won't play anything.

You may also drop an optional logo image or background texture here if you
want to replace the CSS-generated versions later.
